# Stellar Store — Completed

Stellar Store is a production-ready e-commerce frontend built with React, TypeScript, and Vite. Development is complete and the project is integrated with Supabase for authentication, database, and file storage.

Key details:

- Status: Completed and ready for delivery and testing.
- Primary integration: Supabase (Authentication, PostgreSQL database, Storage for images/files).
- Main features: authentication (login/register), product listing, product details, shopping cart, checkout (payment & shipping flow), and user profile.

Local setup:

1. Add your Supabase environment variables to a `.env` file (Supabase URL, ANON KEY, etc.).
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

Notes about Supabase:

- Connection configuration is in `src/config/supabase.ts`.
- Supabase is used for user data storage, session management, and product image storage.
